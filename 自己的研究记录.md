# 环境安装
原版项目使用的是python3.6，torch1.7这个版本太老了。
下面使用的是python3.10对应的install_requirements.sh也做了修改。
其中注意涉及到部分apt安装的部分需要手工sudo安装。
以及部分包只能从官方pip源获取，有的只能通过conda安装。
```
conda create -n mixformer2 python=3.10.18
conda activate mixformer2
bash install_requirements.sh
```
# 导出onnx模型
模型版本要小于19，因为rknn-toolkit2不支持opset19以上版本，这里我们设置成11，同时在导出时做了simplified处理。
```
python export_onnx_4.py \
    --checkpoint ./models/mixformerv2_small.pth.tar \
    --output_dir out_4 \
    --template_size 112 \
    --search_size 224 \
    --tracker_name mixformer2_vit_online \
    --config_name 224_depth4_mlp1_score \
    --opset_version 11
```
